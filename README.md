# 🚀 Projetos-Iniciante-Python

Repositório criado para organizar meus projetos desenvolvidos durante meus estudos em Python.  

Este espaço reúne aplicações práticas, exercícios e pequenos sistemas que desenvolvi com o objetivo de fortalecer minha lógica de programação, melhorar a organização do código e aplicar conceitos fundamentais da linguagem.

Cada pasta representa um projeto independente, permitindo acompanhar minha evolução como desenvolvedor ao longo do tempo.

---

## 📂 Estrutura

Cada diretório contém um projeto específico com seus respectivos arquivos e documentação própria.

Exemplo:

```
Projetos-Iniciante-Python/
│
├── Interest_Calculator/
│   ├── main.py
│   └── README.md
│
└── (novos projetos serão adicionados)
```
